Properties and Tools

Properties:
  • Edit Metadata -> Change Name/Author of map.
  • Edit Scatter Data -> Customize the ghosts scatter data.
  • Default Scatter Data -> Use the default Pacman scatter data modified for your current map.
Tools:
  • Fill Empty Areas with Food -> This will fill all cells that are fillable and empty from y=1 to y=n - 1 (n is max y) and from x=1 to x=m - 1 ( m is max x).
  • Fix Invalid Teleporters by Conversion -> Will convert invalid teleporters to walls.
  • Fix Invalid Teleporters by Removal -> Will remove walls that block teleporters.
  • Clean Useless Properties -> Cleans modifiers/properties that are can not be applied anymore because they will never be reached
