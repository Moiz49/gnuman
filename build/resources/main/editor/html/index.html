General

File:
  • New -> Create a new map. Author/Name must not be filled, size in blocks min. 15, max. 125.
  • Open -> Open a map. Only if valid.
  • Save -> Save a map that has been loaded.
  • Save as -> Save a map under a file name.
  • Reset -> Reset a map to initial state. This can't be undone.
Edit:
  • Redo -> Redo an operation after using Undo (Only for block operations).
  • Undo -> Undo an operation (Only for block operations).
Toolbar:
  • Selection -> Select a block for line tool/draw tool.
  • Draw Tool -> Draw blocks by either dragging or clicking.
  • Line Tool -> Horizontal/Vertical results in lines, diagonal fills a rectangle.
  • Eraser -> Delete blocks by dragging or clicking. Mass erase, select empty and use the line tool.
  • Modifier -> Apply modifiers to a block.
  • Validator -> Check if the map is valid and see the issues it has.
  • Inspector -> Click on a block to see its modifiers.
  • A* Tool -> Click on two points and see if the A* Algorithm can move there. Useful to check if the map is correctly made.
